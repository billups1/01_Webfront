script 언어란?

        기본 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 해주는 용도의 언어

        별도의 소스코드 컴파일을 하지 않고,
        인터프리터(한줄씩 코드를 읽어서 해석하는 기계, 기능)를 이용해서 코드를 바로 실행
        (컴파일 방식은 한번에 묶어서 해석)

        * 장점: 컴파일에 소요되는 시간이 없음
        * 단점: 프로그램 실행 시간이 상대적으로 느림

JavaScript 란?

        웹 브라우저(크롬, 엣지, 웨일, 파이어폭스)에서 주로 사용하는 
        인터프리터 방식의 객체 지향 프로그래밍 언어

JavaScript 코드 작성 방법

        1. 브라우저 개발자도구 -> 콘솔 직접 작성
        2. HTML 내부에 script 태그릴 이용해서 작성(Internal 방식)
