input: inline 형식이다.

    
text 관련 input 태그의 공용속성

    - size: 입력상자 크기
    - maxlength: 입력 받는 텍스트의 최대크기
    - placeholder: 입력상자의 내용 흐리게 보여줌






숫자관련 input 태그 중 공용을 쓸 수 있는 것
- min max : 최대값 최소값
- step : 증가/감소 단계 지정
- range : 슬라이드바를 이용해서 숫자를 지정 (JS와 함께 사용)

폼태그

    1 action: 내부에 작성될 값을 전달할 서버 또는 페이지 주소를 작성
    2 name: form  태그의 이름을 지정하여 폼태그 구분
    3 method: post, get...
    4 target: action에 지정된 주소를 현재탭/새탭 중 어디서 열지 지정
